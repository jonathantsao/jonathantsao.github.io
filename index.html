About Me

I'm a passionate software developer with a sharp mind for problem-solving and conceptualizing tough puzzles. As a quick learner, I use challenges as necessities that push me to maintain an open learning mind. I have ample experience building dynamic web applications and data visualizations using Rails, JavaScript, and React. Furthermore, I'm no stranger to backend languages like Python and Ruby.

There's no other feeling like seeing a project transform from simple wireframes and whiteboard concept maps into a seamless, aesthetically engaging product. At the forefront of all my productions, I set high expectations to push my own limits and ensure that I am always growing as a developer. I'm known for diving too deep, but also as a great team player.

When I'm away from the keyboard, I'm either exploring the great city with my DSLR, playing guitar and singing with friends, or in a local pick-up volleyball game.

Personal Projects

ShareTube

ShareTube is a full-stack web application modeled after YouTube. It is built using Ruby on Rails, a Postgresql database, and React.js with a Redux framework on the frontend.

Live Site Github

Sushi Ninja

SushiNinja is a fun single player game for sushi chefs of all ages. Players have three lives and click and drag to swing their blade. The game is built entirely using JavaScript with the help of processing library, p5. The Sushi Ninja was inspired by HalfBrick's Fruit Ninja.

Live Site Github

Senro

Senro is a Controller and Views framework built on Ruby that uses a Postgresql database. It can integrate Rack middleware and create views in response to HTTP requests. Senro was inspired by Ruby on Rails, and the word "Senro" is a japanese term for "train line".
